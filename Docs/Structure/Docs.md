# Docs 文件夹结构文档

## 概述
Docs 文件夹是项目的文档根目录，包含设计文档、技术文档、用户手册等所有项目相关文档。

## 目录结构

### Design/
- **用途**: 存放系统设计文档
- **内容**:
  - 核心玩法设计文档.md: 游戏核心玩法设计
  - 游戏流程设计文档.md: 游戏流程设计
  - 核心系统设计文档.md: 核心系统架构设计
  - 战斗系统设计文档.md: 战斗系统详细设计
  - 捕捉系统设计文档.md: 捕捉系统详细设计
  - 进化系统设计文档.md: 进化系统详细设计
  - 融合系统设计文档.md: 融合系统详细设计
  - 装备系统设计文档.md: 装备系统详细设计
  - 建筑系统设计文档.md: 建筑系统详细设计

### Structure/
- **用途**: 存放项目结构文档
- **内容**:
  - Assets.md: Assets 文件夹结构说明
  - Src.md: Src 文件夹结构说明
  - Docs.md: Docs 文件夹结构说明
  - Tests.md: Tests 文件夹结构说明
  - Build.md: Build 文件夹结构说明

### Technical/
- **用途**: 存放技术文档
- **内容**:
  - Architecture.md: 系统架构说明
  - CodingStandards.md: 编码规范
  - APIReference.md: API 参考文档
  - PerformanceGuide.md: 性能优化指南

### User/
- **用途**: 存放用户文档
- **内容**:
  - Manual.md: 用户手册
  - FAQ.md: 常见问题解答
  - Tutorials.md: 教程文档

## 设计方案
- 文档分类清晰，便于查找和维护
- 遵循 Markdown 格式，便于版本控制
- 包含设计决策记录，便于后续维护
