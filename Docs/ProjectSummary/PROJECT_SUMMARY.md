# KuKuWorld 项目总结报告

## 项目概述
KuKuWorld 是一款结合了Kuku收集、塔防战斗、建造升级、融合进化的RPG塔防游戏。玩家扮演一位英雄，通过捕捉和培养Kuku，建立防御工事，抵御不断来袭的敌人，守护重要据点。

## 核心玩法
游戏采用创新的"前5分钟捕捉→后续防守"双阶段玩法：

### 捕捉阶段（前5分钟）
- 玩家化身英雄进入新地图
- 地图随机刷新野怪和守护点
- 玩家捕捉随机出现的Kuku
- 野怪死亡后掉落灵魂、金钱和Kuku蛋
- 初级Kuku可吸收核心灵魂进行进化

### 防守阶段（5分钟后）
- Kuku开始不断进攻守护点
- 玩家使用捕捉到的Kuku守护守护点
- 通过建筑生产炮台、机器人、坦克来防守
- 机器人需要通过研究类建筑研究，然后升级
- Kuku进化到最终等级能和特定机器人融合
- 融合后可装备六件装备，可通过商店购买

## 技术架构

### 数据层
- **KukuData.cs**: 基础KuKu数据结构
- **MythicalKukuData.cs**: 神话KuKu数据结构（天界、仙人、神兽、圣者、元始五种稀有度）
- **PlayerData.cs**: 玩家数据管理
- **WeaponData.cs**: 武器装备数据
- **BuildingData.cs**: 建筑数据
- **UnitData.cs**: 游戏单位数据

### 系统层
- **CaptureSystem.cs**: KuKu捕捉系统
- **BattleSystem.cs**: 战斗系统
- **BuildingManager.cs**: 建筑管理系统
- **EvolutionSystem.cs**: KuKu进化系统
- **FusionSystem.cs**: 融合系统
- **EquipmentSystem.cs**: 装备系统
- **GameManager.cs**: 游戏管理器
- **KukuCollectionSystem.cs**: KuKu收集系统

### UI层
- **KukuUIManager.cs**: 通用UI管理

- **ShopUIController.cs**: 商店界面控制器
- **FusionUIController.cs**: 融合界面控制器

### 控制器层
- **MainGameController.cs**: 主游戏控制器

### 工具类
- **KukuGameUtils.cs**: 游戏工具函数库
- **EnemyController.cs**: 敌人AI控制器

## 设计特色

### 神话主题
游戏采用中国神话背景，KuKu具有深厚的文化底蕴，分为五个稀有度等级：
1. 天界 (Celestial) - 蓝色
2. 仙人 (Immortal) - 青色
3. 神兽 (Divine Beast) - 洋红色
4. 圣者 (Sacred) - 黄色
5. 元始 (Primordial) - 红色

### 多样化单位
- 战士类：近战攻击，高防御
- 法师类：远程魔法攻击，区域伤害
- 弓箭手类：远程物理攻击，高精准
- 机器人：科技单位，可升级
- 坦克：重型单位，极高防御
- 飞行单位：无视地形，空中攻击

### 深度融合系统
- KuKu可与机器人融合，获得新能力
- 融合后单位可装备最多六件装备
- 装备可通过商店购买获得
