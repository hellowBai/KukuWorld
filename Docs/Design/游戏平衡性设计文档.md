# KukuWorld 游戏平衡性设计文档

## 1. 平衡性概述

KukuWorld 游戏平衡性设计旨在确保"前5分钟捕捉→后续防守"双阶段玩法的趣味性和挑战性。平衡性设计涵盖资源获取、单位强度、进化速度、融合效果等多个方面，确保玩家在不同阶段都有明确的目标和挑战。

## 2. 资源平衡性

### 2.1 资源类型与获取

#### 2.1.1 金币 (Coin)
- **用途**: 建造建筑、升级单位、购买基础装备
- **捕捉阶段获取**:
  - 普通野生KuKu: 5-15金币
  - 稀有野生KuKu: 15-30金币
  - 野怪死亡掉落: 3-8金币
- **防守阶段获取**:
  - 击败敌人: 8-20金币
  - 完成波次奖励: 20-50金币递增

#### 2.1.2 神石 (Gem)
- **用途**: 解锁高级建筑、购买稀有装备、快速升级
- **捕捉阶段获取**:
  - 稀有野生KuKu: 1-3神石
  - 特殊事件: 2-5神石
- **防守阶段获取**:
  - 击败精英敌人: 2-4神石
  - 完成困难波次: 3-8神石

#### 2.1.3 灵魂 (Soul)
- **用途**: KuKu进化、属性提升、技能解锁
- **获取方式**:
  - 野生KuKu死亡: 0.5-2.0灵魂值
  - 敌人死亡: 0.3-1.5灵魂值
  - 完成挑战: 2.0-5.0灵魂值

### 2.2 资源平衡公式

#### 2.2.1 金币获取平衡
```
基础金币 = (敌人等级 × 2) + 随机波动(±20%)
捕捉阶段总金币 ≈ 防守阶段总金币 × 0.7
```

#### 2.2.2 神石获取平衡
```
基础神石 = 敌人稀有度系数 × 0.5
捕捉阶段神石 ≈ 防守阶段神石 × 0.5
```

#### 2.2.3 灵魂获取平衡
```
基础灵魂 = (敌人等级 × 0.3) × 稀有度系数
总灵魂需求 = 进化等级² × 10
```

## 3. KuKu平衡性

### 3.1 稀有度平衡

#### 3.1.1 稀有度属性倍数
| 稀有度 | 攻击力倍数 | 防御力倍数 | 生命值倍数 | 捕捉难度 | 进化效率 |
|--------|------------|------------|------------|----------|----------|
| 天界 (Celestial) | 1.0x | 1.0x | 1.0x | 1.0x | 1.0x |
| 仙人 (Immortal) | 1.3x | 1.2x | 1.4x | 1.2x | 1.3x |
| 神兽 (DivineBeast) | 1.7x | 1.5x | 1.8x | 1.5x | 1.6x |
| 圣者 (Sacred) | 2.2x | 2.0x | 2.5x | 1.8x | 2.0x |
| 元始 (Primordial) | 3.0x | 2.8x | 3.5x | 2.5x | 2.8x |

#### 3.1.2 生成概率
```
天界: 40%
仙人: 30%
神兽: 15%
圣者: 10%
元始: 5%
```

### 3.2 属性平衡公式

#### 3.2.1 基础属性生成
```
基础攻击力 = 20 + (等级 × 5) × 稀有度倍数
基础防御力 = 10 + (等级 × 3) × 稀有度倍数
基础生命值 = 50 + (等级 × 10) × 稀有度倍数
基础速度 = 2.0 + (等级 × 0.1) × 稀有度倍数
```

#### 3.2.2 进化属性增长
```
进化后属性 = 当前属性 × (1 + 进化等级 × 0.2)
最大进化等级 = 5
```

## 4. 捕捉机制平衡

### 4.1 捕捉成功率计算
```
基础成功率 = 0.8
生命值修正 = (1 - 当前生命值/最大生命值) × 0.5
稀有度修正 = 1 - 稀有度系数 × 0.15
捕捉者修正 = 捕捉者等级/100 × 0.2
环境修正 = 随机值(-0.1 到 +0.1)

最终成功率 = 基础成功率 + 生命值修正 + 稀有度修正 + 捕捉者修正 + 环境修正
最终成功率限制在 0.05 到 0.95 之间
```

### 4.2 捕捉难度调整
- **简单难度**: 成功率增加 20%
- **标准难度**: 默认平衡
- **困难难度**: 成功率减少 20%

## 5. 战斗平衡性

### 5.1 伤害计算公式
```
基础伤害 = 攻击方攻击力 × (1 - 防御方防御力/(防御方防御力 + 100))
元素相克修正 = ±25%
暴击修正 = ×2.0 (10%概率)
```

### 5.2 敌人强度平衡

#### 5.2.1 敌人生成规律
```
第N波敌人强度 = 基础强度 × (1 + N × 0.15)
精英敌人出现频率 = 每5波出现1个
Boss敌人出现频率 = 每10波出现1个
```

#### 5.2.2 敌人属性倍数
| 敌人类型 | 攻击力倍数 | 生命值倍数 | 速度倍数 | 特殊能力 |
|----------|------------|------------|----------|----------|
| 普通敌人 | 1.0x | 1.0x | 1.0x | 无 |
| 精英敌人 | 2.0x | 2.5x | 1.2x | 震晕攻击 |
| Boss敌人 | 3.5x | 5.0x | 1.5x | 群体攻击 |

### 5.3 防守单位平衡

#### 5.3.1 建筑产出平衡
```
炮台成本: 50金币, 输出: 10 DPS
机器人成本: 100金币 + 1神石, 输出: 25 DPS, 生命: 100
坦克成本: 200金币 + 2神石, 输出: 50 DPS, 生命: 300, 防御: 20
```

## 6. 进化系统平衡

### 6.1 进化需求计算
```
第N级进化所需灵魂 = 初始灵魂 × (进化等级²) × 稀有度系数
天界: 系数 = 1.0
仙人: 系数 = 1.5
神兽: 系数 = 2.0
圣者: 系数 = 3.0
元始: 系数 = 5.0
```

### 6.2 进化收益
```
每级进化属性提升 = 当前属性 × 0.2
技能解锁: 第3级解锁特殊技能
最大进化等级: 5级
```

## 7. 融合系统平衡

### 7.1 融合条件
- KuKu必须达到5级进化
- 机器人必须达到10级
- 需要50灵魂和10神石
- 融合兼容性 ≥ 0.7

### 7.2 融合效果计算
```
融合后攻击力 = (KuKu攻击力 × 0.6) + (机器人攻击力 × 0.4) × 兼容性系数
融合后生命值 = (KuKu生命值 × 0.4) + (机器人生命值 × 0.6) × 兼容性系数
融合后获得6个装备槽位
```

### 7.3 兼容性计算
```
兼容性 = (元素匹配度 × 0.4) + (等级差异修正 × 0.3) + (稀有度匹配 × 0.3)
元素匹配度: 相同元素 = 1.0, 相关元素 = 0.8, 不相关 = 0.5
等级差异修正: 差异≤2级 = 1.0, 差异≤5级 = 0.8, >5级 = 0.5
```

## 9. 阶段平衡性

### 9.1 捕捉阶段平衡
- **时间压力**: 5分钟限制创造紧迫感
- **资源分配**: 捕捉效率 vs 资源获取的权衡
- **风险回报**: 高稀有度KuKu需要更多投入

### 9.2 防守阶段平衡
- **难度曲线**: 逐渐增加的敌人强度
- **策略深度**: 多样化的防守单位组合
- **资源管理**: 有限资源下的最优配置

### 9.3 阶段转换平衡
```
捕捉阶段资源 ≈ 防守阶段初期需求 × 0.8
给玩家留出调整和优化空间
```

## 10. 难度调节机制

### 10.1 动态难度调整
```
如果玩家连续失败3次: 敌人强度减少 10%
如果玩家连续成功5次: 敌人强度增加 5%
```

### 10.2 辅助机制
- **新手保护**: 前3次游戏降低敌人强度
- **失败补偿**: 连续失败后增加资源奖励
- **进度奖励**: 根据游戏进度调整奖励

## 11. 长期游戏平衡

### 11.1 内容消耗平衡
```
每日可获取资源 ≈ 每日推荐消费
每周挑战内容 = 10小时游戏量
每月新增内容 = 20小时游戏量
```

### 11.2 重复游戏性
- 随机生成的地图
- 不同的KuKu组合
- 多样的策略选择
- 成就系统激励
