# KukuWorld - Unity集成版

![KukuWorld Logo](Sprites/kukuworld_logo.png)

**KukuWorld** 是一款结合了Kuku收集、塔防战斗、建造升级、融合进化的RPG塔防游戏。现已成功集成到Unity游戏开发框架中，实现了完整的功能和优化的性能。

## 🎮 游戏特色

### 独特玩法机制
- **双阶段游戏**: "前5分钟捕捉→后续防守"的创新玩法
- **神话主题**: 中国神话背景的Kuku设计
- **深度进化**: 通过灵魂吸收实现Kuku进化
- **复杂融合**: Kuku与机器人融合产生新单位
- **策略塔防**: 丰富的建筑和装备系统

### 稀有度系统
- 🌟 **天界 (Celestial)** - 蓝色品质
- 🌊 **仙人 (Immortal)** - 青色品质  
- 🐉 **神兽 (Divine Beast)** - 洋红色品质
- ⚡ **圣者 (Sacred)** - 黄色品质
- 🔥 **元始 (Primordial)** - 红色品质

## 🏗️ 技术架构

### 核心系统
- **捕捉系统**: 野生Kuku生成、捕捉逻辑、奖励机制
- **战斗系统**: 双阶段战斗模式、单位AI、伤害计算
- **进化系统**: 灵魂吸收、属性提升、进化逻辑
- **融合系统**: Kuku与机器人融合、属性合成
- **装备系统**: 6件装备槽位、商店购买
- **建筑系统**: 生产和防御建筑、升级机制

### Unity集成特性
- **场景管理**: 完整的场景切换系统
- **UI系统**: Unity UI Toolkit驱动
- **音频系统**: 背景音乐和音效管理
- **事件系统**: Unity事件驱动架构
- **资源管理**: 优化的资源加载系统

## 📁 项目结构

```
Assets/
├── Scripts/                 # 游戏逻辑脚本
│   ├── Core/               # 核心系统
│   ├── Data/               # 数据模型
│   ├── Systems/            # 游戏系统
│   ├── UI/                 # 用户界面
│   ├── Controllers/        # 游戏控制器
│   └── Utils/              # 工具类
├── Prefabs/                # 预制体
├── Scenes/                 # Unity场景
├── Resources/              # 资源
├── Sprites/                # 精灵图像
├── Audio/                  # 音频文件
└── Materials/              # 材质
```

## 🚀 快速开始

### 环境要求
- Unity 2022.3 LTS 或更高版本
- Windows 10+, macOS 10.15+, 或 Ubuntu 18.04+
- 4GB RAM 最低，推荐 8GB+

### 运行项目
1. 克隆或下载项目
2. 使用Unity Hub打开项目
3. 等待资源导入完成
4. 运行 `Scenes/MainMenu.unity` 场景

### 开发指南
- 核心逻辑位于 `Scripts/` 目录
- 游戏数据定义在 `Scripts/Data/` 目录
- 系统实现在 `Scripts/Systems/` 目录
- UI逻辑在 `Scripts/UI/` 目录
